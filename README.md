# Olga Geometry

Responsive landing page for Olga Geometry mandala courses.

## Run locally

Open `index.html` directly, or serve the directory with any static web server.

## Deploy

The project is a static site and can be deployed on GitHub Pages, Vercel, Netlify, Cloudflare Pages, or any static hosting.

### GitHub Pages

Repository Settings → Pages → Build and deployment → Deploy from a branch → `main` / root.

## Structure

- `index.html` — landing page and product architecture
- `styles.css` — responsive visual system

Current image backgrounds use remote Unsplash assets as temporary placeholders. Replace them with Olga Geometry's original photography before production launch.
